piPowerTempLog  

A sub project to monitor power and temperatures

==========
>$ sudo ./piPowerTempLogInstall.sh

Setting up the arduino power poller
==========
Build Arduino shield according to files in  

	Documents

Compile and upload files found in  

	ardPowerTempLog
	
to Arduino
